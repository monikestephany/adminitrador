	1.Dados do template

		1.1 Campo Descrição da unidade organizacional
			Criar uma lista fixa, que será populada com as unidades selecionadas. Deverá ser considerado que poderá haver muitos registros, verificar a necessidade de uma barra de rolagem ou opção de visualizar que ao ser acionada exibirá todas as unidades.

		1.2 Local da revista
			Deverá ter uma opção para replicar o local da unidade. Caso existam mais de uma unidade, deverá exibir uma opção para permitir selecionar de qual unidade será replicado o local da revista. Deverá permitir também que seja preenchido qualquer informação no campo.

	2.Tipo de Regime

		2.1 Preleção: Tempo previsto para passar aos PM que estarão de serviço as informações/instruções da execução de uma escala. Os valores desse campo devem ser de 5 minutos até 30 minutos, com um intervalo de 5 minutos.

		2.2 Trabalha no feriado: Indica se irá utilizar o cadastro de feriado para geração de escala Nota: Apresentado apenas para regime de trabalho 5x2 e 6x1

		2.3 Quantidade de pessoas para a escala: Terá um campo para informar a quantidade de efetivo necessário para a escala, para que no momento de elaboração seja criado um controle do efetivo necessário.

		2.4Criar configuração por Posto/Graduação: Deverá permitir qual Posto/Graduação irá compor a escala, para que no momento de elaboração seja criado uma limitação por patente.
